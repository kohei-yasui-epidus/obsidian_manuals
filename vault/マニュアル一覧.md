# 📖 マニュアル一覧

全マニュアルの一覧です。列見出しをクリックすると並べ替えできます。

## 公開中

```dataview
TABLE WITHOUT ID file.link AS "マニュアル", category AS "カテゴリ", created AS "作成日", updated AS "更新日"
FROM "マニュアル"
WHERE status != "アーカイブ" AND status != "下書き"
SORT category ASC, file.name ASC
```

## 下書き(作成中)

```dataview
TABLE WITHOUT ID file.link AS "マニュアル", category AS "カテゴリ", updated AS "更新日"
FROM "マニュアル"
WHERE status = "下書き"
SORT updated DESC
```

## アーカイブ(旧版・廃止)

```dataview
TABLE WITHOUT ID file.link AS "マニュアル", category AS "カテゴリ", updated AS "更新日"
FROM "マニュアル"
WHERE status = "アーカイブ"
SORT updated DESC
```
