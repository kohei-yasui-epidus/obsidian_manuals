# はじめに(閲覧メンバー向けセットアップ)

このVaultはチームのマニュアルをObsidianで閲覧・管理するためのものです。
**閲覧するだけなら以下の3ステップで完了します(所要 約10分)。**

## 1. Obsidian をインストール

- https://obsidian.md からダウンロードしてインストール(無料)

## 2. リポジトリを clone

ターミナルで:

```bash
git clone <このリポジトリのURL> obsidian_manuals
```

> [!note] Gitが初めての方へ
> 管理者に声をかけてください。clone用のURLとアクセス権を案内します。

## 3. Obsidian で Vault を開く

1. Obsidian を起動 →「**フォルダをVaultとして開く**」
2. clone した `obsidian_manuals` の中の **`vault` フォルダ** を選択(リポジトリ直下ではなく `vault/` を選ぶこと)
3. 「制限モード(Restricted mode)を無効にする」を選び、コミュニティプラグインを**信頼して有効化**する
   - Dataview / Obsidian Git / Omnisearch / Homepage / Recent Files が同梱済みで、設定も共有されています

これで起動時に **[[Home]](マニュアルポータル)** が自動で開きます。

## 日々の使い方

| やりたいこと | 操作 |
| --- | --- |
| 全文検索 | `Cmd + Shift + F` |
| タイトルで開く | `Cmd + O` |
| 新着・通達を確認 | [[Home]] を開く(起動時に自動表示) |
| 最新版に更新 | 自動(10分ごとにgit pull)。手動なら `Cmd + P` →「Git: Pull」 |

## 困ったとき

- 表示が変・一覧が出ない → `Cmd + P` →「Reload app without saving」
- それでも直らない → 管理者([[チーム運用ルール(Git同期)]] 参照)へ
